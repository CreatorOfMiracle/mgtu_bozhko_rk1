# mgtu_bozhko_lab1

## Описание

Этот проект — веб-приложение на Next.js с использованием Tailwind CSS.

## Запуск проекта

1. Установите зависимости:

```bash
npm install
```

2. Запустите сервер разработки:

```bash
npm run dev
```

3. Откройте браузер и перейдите по адресу:

```
http://localhost:3000
```

## Сборка для продакшена

```bash
npm run build
npm start
```

## Стек технологий
- Next.js
- React
